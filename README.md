What This Site Is
Civil Rights Hub is a civic technology platform built by We The People News- a civil rights journalism
and activism organization. The platform aims to be the nation’s most comprehensive civil rights resource,
providing tools for activists, journalists, attorneys, and everyday citizens to know their rights, document
violations, access legal help, and stay informed.
The site is entirely free, carries no advertisements, and operates on donations via Patreon, CashApp, and
Venmo.

Critical Issues Identified
HIGH PRIORITY (Affecting Credibility & Trust)
1. Attorney directory appears empty when filtered- contradicts “1,756 attorneys” stat
2. Violation feed shows 0 recent activity- makes platform look abandoned
3. Only 3 visible community members- network effect not achieved
4. FOIA tracker shows “No activity yet”- no demonstration of value
5. Location defaults to Jakarta for non-US visitors- many features require US location context
MEDIUM PRIORITY (Functionality Gaps)
6. Many tools appear to be UI shells without deep backend data
7. Newsroom has only ~5 articles- not enough to be a regular destination
8. City pages appear templated with similar content
9. No clear About page- who runs this, what’s the mission, who’s behind it
10. Help Now button in corner- purpose unclear
11. Newsletter requires sign-in- friction for casual visitors
LOW PRIORITY (Enhancement Opportunities)
12. No volunteer/contribute page beyond “Join Network”
13. No impact metrics or success stories
14. No mobile app
15. Only English- no multilingual support
16. No accessibility statement
17. SEO likely needs optimization (limited search presence)
18. No open source repository link despite “Open Source” badge
19. No analytics dashboard for public transparency
20. Missing integrations with major civil rights orgs (ACLU, EFF, etc. beyond footer links)
What’s Working Well
• Clean, modern UI design with good visual hierarchy
• Location-aware personalization concept is strong
• Resource Command searchable menu is innovative
• Constitutional rights cards are clear and accessible
• Scanner feed integration works (500 active feeds)
• Authentication system exists with gamification (Civic Score)
• Footer city hub structure is good for local SEO
• Trust signals present: “Educational resource- not a substitute for legal advice”
• Strong brand identity: “Your Rights. Defended.” tagline

Recommended Priority Focus Areas
1. Populate the Data (Weeks 1-4)
The #1 issue is empty states. Every tool that shows “No activity” or “No results” erodes trust.
• Import public attorney databases (state bar associations)
• Seed violation reports from public records
• Add FOIA template library
• Populate state statute database
2. Fix the Empty States (Weeks 2-4)
• Show demo/example data for first-time visitors
• Add “Recently Added” sections
• Create compelling empty-state CTAs
• Add sample reports and cases
3. Content & SEO (Ongoing)
• Publish 2-3 newsroom articles per week
• Create city-specific resource pages
• Add comprehensive state law guides
• Build out the incident guide library
